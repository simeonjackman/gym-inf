## Arbeiten mit KI

Seit der Erfindung von LLMs (Large Language Models) sind viele Aufgaben in der
Schule deutlich einfacher geworden. Man kann gewisse Hausaufgaben sehr einfach
von einer KI schreiben lassen, oder sogar ganze Aufsätze von einer KI erstellen
lassen. Auch Code kann man relativ gut von einer KI erzeugen lassen.

Die KIs scheinen auch recht gut darin zu sein, grössere Codestücke zu schreiben.
Hier ist jedoch Vorsicht geboten, denn Code, der von einer KI geschrieben wurde,
hat keine Garantie auf Korrektheit.

Wir können die KI dennoch nutzen um uns zu helfen, wir sollten uns dabei einfach
an einige Grundregeln halten.

      <Fragment>
        ```text
        Wie kann ich alle e's aus einem Text löschen?
        ```
      </Fragment>
      <Fragment>
        Fragen Sie Codeium wie man diese Aufgabe lösen kann, die KI wird Ihnen dann sogar Code zur Lösung dazu geben.
      </Fragment>
    </Basement>
    <Basement>
      
```text
Um alle "e"s aus einem Text zu löschen, kannst du die
replace-Methode verwenden.
```
      </Basement>
      <Basement>
       Wenn Sie nicht verstehen wie das geht, oder es noch nicht in Ihr aktuelles Wissen passt, dann können Sie die KI auch nach einer anderen Formulierung der Aufgabe fragen.
      </Basement>
      <Basement>
      ```text
      Wie kann ich das mit schleifen machen?
      ```
      
    </Basement>
    <Basement>
        ```text
        Um alle "e"s aus einem Text mit einer Schleife
        zu entfernen, kannst du die einzelnen Zeichen 
        des Textes überprüfen und nur diejenigen 
        hinzufügen, die nicht "e" sind. 
        ```
    </Basement>
    <Basement>
      Sie können dann auch direkt nach dem Code fragen. Dabei sollten Sie dann aber die Programmiersprache angeben, in der Sie arbeiten.
      </Basement>
      <Basement>
      ```javascript
      let text = "Dies ist ein Beispieltext";
      let textWithoutE = "";
      for (let i = 0; i < text.length; i++) {
          if (text[i] !== "e") {
              textWithoutE += text[i];
          }
      }

      console.log(textWithoutE);
      ```
     </Basement>
     <Basement>
        Dieser Code muss dann normalerweise noch an das eigene Projekt angepasst werden. Es kann jedoch eine gute Starthilfe sein.
     </Basement>
