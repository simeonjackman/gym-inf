## Installation

<section>

In der Informatik braucht es viele verschiedenen Werkzeuge, um mit dem Computer
und auch mit Code zu arbeiten. Diese Werkzeuge müssen zuerst installiert werden.
Im folgenden Abschnitt finden Sie eine Installationsanleitung für Ihr
Betriebssystem. Stellen Sie sicher dass Sie das richtige Betriebssystem
ausgewählt haben, ansonsten können Sie die Installation nicht korrekt
durchführen.

<TabContainer>
  <ul className="tabs">
    <TabsButton isActive={true} tabIndex="os-windows" label="Windows" />
    <TabsButton tabIndex="os-mac" label="MacOS" />
  </ul>
  <Tab className="show" tabIndex="os-mac">
    Wenn Sie einen Mac haben, können Sie viele der Programme am über `homebrew` installieren. Dafür müssen Sie aber zuerst `homebrew` selber installieren. Das geht ganz einfach wenn Sie mit `cmd + Space` eine Suche öffnen, und nach dem Programm `Terminal` suchen. Dann einfach den folgenden Befehl hineinkopieren und auf `Enter` drücken. Mehr Informationen zu `homebrew` finden Sie [hier](https://brew.sh/), falls Sie diese brauchen.

    ```bash
    /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
    ```

    ### VSCode

    Dann installieren wir VSCode. Das ist unser Editor, den wir zum Schreiben und Editieren von Code verwenden werden. Um VSCode zu installieren, geben Sie den folgenden Befehl ein, und drücken Sie anschliessend auf `Enter`. Stellen Sie sicher, dass der Befehl richtig geschrieben ist, ansonsten wird es nicht funktionieren.

    ```bash
    brew install --cask visual-studio-code
    ```

    ### Git (Versionierungs-Werkzeug)

    Wenn man Code schreibt, möchte man diesen immer in Versionen verwalten. Dafür gibt es das Werkzeug `git` dass Sie ganz einfach mit dem folgenden Befehl installieren können. Wir benötigen `git` auch um die Projektvorlagen herunterzuladen und auch um diese abzugeben.

    ```bash
    brew install git
    ```

    ### Chrome (Webbrowser)

    [**Falls nicht bereits intalliert**] Wir werden sehr viel über die Webentwicklung lernen, daher brauchen wir einen Browser mit einem modernen Standart. Dafür können Sie `Chrome` verwenden, es gehen aber auch andere Browser wie Edge, Brave, Firefox etc. Getestet ist der ganze Code aber nur auf Browsern die auf der Chromium-Engine basieren. Das sollte aber keine Probleme machen. Chrome kann auch mit dem einfachen Befehl hier installiert werden.

    ```bash
    brew install --cask google-chrome
    ```

    ### Live Server

    Moderne Webseiten brauchen jeweils einen Webserver um zu funktionieren. Dafür
    verwenden wir `Live Server`. Dazu müssen wir eine **Erweiterung** in `VSCode`
    installieren. Öffnen Sie Visual Studio Code (VSCode), drücken Sie dann
    `Command + P` und kopieren Sie den folgenden Text in das Feld und drücken Sie
    `Enter`.

    ```text
    ext install ritwickdey.liveserver
    ```

    Wählen Sie dann die Extension `Live Server` auf der linken Seite und drücken Sie
    auf install.

    Am einfachsten geht die Installation auf Windows mit dem Kommandozeilen-Befehl `winget`. Dafür müssen Sie eine neue Powershell öffnen. Das machen Sie am besten als Administrator, damit die Installation auch sicher funktioniert.

    Öffnen Sie das Startmenu und suchen Sie nach `powershell`. Mit einem Rechtsklick auf das Programm, können Sie dann `Als Administrator öffnen` auswählen. Dann sollte ein neues Fenster aufgehen, wo Sie Befehle ausführen können.

    Damit Sie `winget` sicher haben, und dieses auch auf der neusten Version ist, können Sie den folgenden Befehl laufen lassen.

    ```bash
    Add-AppxPackage -Path 'https://github.com/microsoft/winget-cli/releases/download/v1.8.1911/Microsoft.DesktopAppInstaller_8wekyb3d8bbwe.msixbundle' -ForceApplicationShutdown
    ```

Um VSCode zu installieren, geben Sie den folgenden Befehl ein, und drücken Sie anschliessend auf `Enter`. Stellen Sie sicher, dass der Befehl richtig geschrieben ist, ansonsten wird es nicht funktionieren.

## Github

<section>
  GitHub ist eine Plattform zur Versionsverwaltung und kollaborativen
  Softwareentwicklung. Sie basiert auf dem Versionskontrollsystem Git und
  ermöglicht es EntwicklerInnen, ihren Code zu speichern, gemeinsam daran zu
  arbeiten und Änderungen nachzuverfolgen. Wir verwenden dieses Tool ab jetzt
  für den Unterricht. Gehen Sie auf [github.com](https://github.com) und
  erstellen Sie mit Ihrer Schul-E-Mailadresse einen Account.
</section>

## Demo Projekt

<section>

Wenn Sie das Projekt erfolgreich geklont haben, haben Sie alle Dateien, sowie
den ganzen Versionsverlauf bei Ihnen auf dem Computer. Sie können im `Explorer`
(ganz links bei `VSCode`) schauen welche Dateien verfügbar sind. Jedes Projekt /
Repository sollte immer eine `README.md` Datei haben, die Sie durchlesen
sollten. Da stehen alle Informationen die Sie zu dem Projekt brauchen. Bei
unseren Projekten, stehen da auch gleich die Aufgaben. Diese Datei sollten Sie
also immer gut durchlesen. Oftmals ist es besser diese Datei direkt auf `Github`
anzuschauen, weil sie dort etwas schöner dargestellt wird. Dafür folgen Sie
einfach diesem Link hier:
[https://github.com/gymmu/00-demo.git](https://github.com/gymmu/00-demo.git) und
scrollen ein wenig nach unten.

</section>

## Webseite anpassen

Bis jetzt haben Sie das Projekt einfach zum laufen gebracht, aber noch nichts
von dem Inhalt geändert. Da es Ihr Projekt ist, soll die Webseite auch etwas
über Sie aussagen.

<Example title="Auftrag: Erste Version">
  Suchen Sie also nach der Datei `index.html` und fügen Sie
  dort Ihren Namen ein. Speichern Sie dann mit `CTRL + S` und schauen Sie sich das
  Resultat im Webbrowser an.
  Herzlichen Glückwunsch! Mit dem Einfügen von Ihrem Namen, haben Sie die erste
  eigene Version von Ihrem Projekt erstellt. Das möchten wir auch so als Version
  festhalten und mithilfe von `git` einen sogenannten `commit` erstellen.
  Fügen Sie als erstes beim `git graph` Ihren Benutzer von Github ein, indem Sie auf das Einstellungssymbol (Zahnrad) drücken.
  Stagen Sie dann die `index.html` Datei. Schreiben Sie eine `commit-message` und drücken Sie auf `Commit`.

</Example>
