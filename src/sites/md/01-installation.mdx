## Installation

<section>

In der Informatik braucht es viele verschiedenen Werkzeuge um mit dem Computer
und auch mit Code zu arbeiten. Diese Werkzeuge müssen zuerst installiert werden.
Im folgenden Abschnitt finden Sie eine Installationsanleitung für Ihr
Betriebssystem. Stellen Sie sicher dass Sie das richtige Betriebssystem
ausgewählt haben, ansonsten können Sie die Installation garnicht durchführen.

    Am einfachsten geht die Instalation auf Windows mit dem Kommandozeilen-Befehl `winget`. Dafür müssen Sie ein neues Terminal öffnen. Das machen Sie am besten als Administrator, damit die Installation auch sicher funktioniert.

    öffnen Sie das Startmenu und suchen Sie nach `cmd`. Mit einem Rechtsklick auf das Programm, können Sie dann `Als Administrator öffnen` auswählen. Dann sollte ein neues Fenster aufgehen, wo Sie Befehle ausführen können.

    ### VSCode

    Als erstes installieren wir VSCode. Das ist unser Editor den wir zum schreiben und editieren von Code verwenden werden. Um VSCode zu installieren, geben Sie den folgenden Befehl ein, und drücken Sie anschliessend auf `Enter`. Stellen Sie sicher das der Befehl richtig geschrieben ist, ansonsten wird es nicht funktionieren.

    ```cmd
    winget install -e --id Microsoft.VisualStudioCode
    ```

    ### Git (Versionierungs-Werkzeug)

    Wenn man Code schreibt, möchte man diesen immer in Versionen verwalten. Dafür gibt es das Werkzeug `git` dass Sie ganz einfach mit dem Befehl hier installieren können. Wir benötigen `git` auch um die Projektvorlagen zu bekommen und auch um diese abzugeben.

    ```cmd
    winget install -e --id Git.Git
    ```

    ### Chrome (Webbrowser)

    Wir werden sehr viel über die Webentwicklung lernen, daher brauchen wir einen Browser mit einem modernen Standart. Dafür können Sie `Chrome` verwenden, es gehen aber auch andere Browser wie Edge, Brave, Firefox etc. Getestet ist der ganze Code aber nur auf Browsern die auf der Chromium-Engine basieren. Das sollte aber keine Probleme machen. Chrome kann auch mit dem einfachen Befehl hier installiert werden.

    ```cmd
    winget install -e --id Google.Chrome
    ```

    ### Node.js (Javascript Runtime)

    Wir werden später viel mit Javascript programmieren, dafür werden wir dann `nodejs` brauchen. Wir brauchen `nodejs` aber bereits jetzt um verschiedene Teile des Projektes zum laufen zu bringen. `Nodejs` kann am besten mit dem folgenden Befehl installiert werden.

    ```cmd
    winget install -e --id OpenJS.NodeJS
    ```

    ### Docker-Desktop

    `Docker` ist ein Werkzeug mit dem man Container erstellen und laufen lassen kann. Das hat den Vorteil dass wir auf verschiedenen Geräten die gleiche Umgebung in einem Container laufen lassen können, somit sollte nach der Installation von all diesen Werkzeugen bei allen alles funktionieren, zumindest wenn all diese Werkzeuge korrekt installiert wurden. `Docker` können Sie auch mit diesem Befehl installieren.

    ```cmd
    winget install -e --id Docker.DockerDesktop
    ```

## Webserver starten

Moderne Webseiten brauchen jeweils einen Webserver um zu funktionieren. Dafür
verwenden wir `Docker` zusammen mit `nodejs`. Damit wir `Docker` direkt
verwenden können, müssen wir eine **Erweiterung** in `VSCode` installieren.
Drücken Sie dafür einfach `CTRL + P` und kopieren Sie den folgenden Text in das
Feld und drücken Sie `Enter`.

```text
ext install ms-vscode-remote.remote-containers
```

Wenn die **Extension** erfolgreich installiert werden konnte, können Sie das
Projekt in einem Container öffnen. Dafür klicken Sie `CTRL + SHIFT + P` und
suchen nach dem Befehl `Reopen in Container`. Wenn Sie diesen Befehl ausführen,
passieren ganz viele magische Dinge, und beim ersten Mal dauert das auch eine
Weile. Wenn der ganze Prozess aber abgeschlossen ist, dann verwenden wir alle
das genau gleiche Projekt-Setup und sollten dannach keine weiteren Probleme mehr
haben.

<Example title="Auftrag: Mit dem Container kommunizieren">
  Wenn Sie mit dem Container kommunizieren möchten, müssen Sie das über ein
  Terminal machen. Das Terminal kann am Anfang ein wenig einschüchternd wirken,
  es ist aber sehr einfach zu bedienen, und auch sehr mächtig. Sie müssen in der
  Regel nur einfache Befehle eingeben, und wenn etwas nicht funktionieren
  sollte, dann bekommen Sie eine ausführliche Fehlermeldung. Wenn Sie aber alles
  richtig geschrieben haben, dann sollte auch alles ohne Probleme funktionieren.

Um ein neues Terminal zu öffnen, tippen Sie `CTRL + SHIFT + P` und suchen Sie
nach dem Befehl `Terminal: Create New Terminal`. Dann sollte am unteren Rand ein
neues Terminal aufgehen. Dort können Sie den Befehl `npm run dev` eintippen und
mit `Enter` bestätigen. Das erstellt Ihnen einen neuen Webserver.

Wenn alles richtig funktioniert hat, können Sie über den folgenden Link Ihre
eigene Webseite anzeigen lassen: [http://localhost:5173/](http://localhost:5173)

</Example>
