## Vektorgrafiken mit SVG

<Example title="Auftrag">

Gehen Sie auf die folgende Webseite:

```text
https://github.com/simeonjackman/01-grafiken-mit-svg.git
```

Drücken Sie dann auf den grünen Knopf `"Use this template"` Drücken Sie dann auf
`"Create a new repository"` Kreieren Sie ein eigenes repository mit dem Namen
01-grafiken-mit-svg.

Wechseln Sie dann zurück ins VSCode und drücken Sie `CTRL + SHIFT + P` und
tippen Sie `clone`. Wählen Sie dann `Git: Clone` aus, und klonen Sie das
folgende Repository zu Ihnen auf den Computer. (ersetzen Sie IHR-GITGUB-NAME mit
Ihrem Github Namen)

```text
https://github.com/IHR-GITGUB-NAME/01-grafiken-mit-svg.git
```

</Example>
