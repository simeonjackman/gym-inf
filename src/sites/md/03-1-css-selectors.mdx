## CSS Selektoren

<section>

CSS ist von der Syntax her eine sehr einfache Sprache. CSS hat Regeln, die immer
im gleichen Style aufgebaut sind.

```css
selector {
  property: value;
}
```

Die Teile in dem Block, sind selbsterklärend. Mit

`property` sagt man welche **Eigenschaft** man verändern möchte, und `value` ist
der **Wert** für diese Eigenschaft. Eine grüne Schriftfarbe erreicht man zum
Beispiel mit `color: green;`.

Mit dem `selector` kann man sagen für welche Elemente die Regeln gelten sollen.
Da haben wir bereits gesehen das man Tags verwenden kann (`<h1>`, `<body>`,
...). Man kann aber auch Klassen und IDs verwenden.

</section>

### CSS Klassen

<section>

Klassen werden vermutlich am häufigsten verwendet in CSS. In HTML werden Klassen
als Attribut gesetzt, wie zum Beispiel:

```html
<p class="important">Sehr wichtig</p>
```

In CSS gibt uns das die Möglichkeit andere Stile für `<p>` -Tags anzugeben, die
ebenfalls die Klasse `wichtig` haben.

</section>

<Example title="Aufgabe: Klassenselektor verwenden">

Erstellen Sie eine neue Regel in Ihrem CSS, die einen Klassenselektor verwendet.
Sie können Sich dabei an dem folgenden Beispiel orientieren.

```css
.important {
  border: 5px solid orange;
}
```

</Example>

Um in CSS zu sagen dass es sich um eine Klasse handelt, müssen Sie einfach einen
Punkt vor den Klassennamen schreiben.

### CSS IDs

Es können nicht nur Klassen sondern auch IDs verwendet werden. Dies wird oftmals
gemacht wenn es sich um ein eindeutiges Element handelt, wie zum Beispiel eine
Navigationsliste.

Ähnlich wie bei Klassen, muss auch nur ein spezielles Zeichen vor den Namen
gestellt werden. Bei IDs wird das Zeichen `#`{" "} verwendet.

<Example title="Aufgabe: IDs verwenden">

Geben Sie einem Element auf Ihrer Webseite eine ID, und erstellen Sie eine Regel
für dieses Element. Sie können Sich dabei an dem folgenden Beispiel orientieren.

```html
<div id="menu-button">
  ...
</div>
```
In CSS (Bsp. im style.css) können Sie diese ID folgendermassen ansteuern:
```css
#menu-button{
  ...
}
```

</Example>

### Pseudo-Klassen

In CSS gibt es bestimmte bereits vordefinierte Pseudo-Klassen. Diese können eine
Element auswählen das zur Zeit gerade den Cursor auf sich hat.

<Example title="Beispiel: Elemente vergrössern">

Um einen Link besser hervorzuheben, können Sie die Schriftgrösse anpassen wenn
der Mauszeiger über dem Link ist.

```css
a:hover {
  font-size: 1.4rem;
}
```

Sie können das `:hover` zu jedem Element dass Sie hervorheben möchten
hinzufügen.

</Example>

Pseudo-Klassen beginnen jeweils mit einem `:`, wie zum Beispiel `:visited`.

<Example title="Aufgabe: Besuchte Links verstecken">

Normalerweise sind Links die Sie bereits besucht haben einfach violet. Versuchen
Sie mit der Pseudo-Klasse `:visited` einen Link zu verstecken.

</Example>
